##  Deskripsi Aplikasi
Aplikasi ini merupakan aplikasi kuis interaktif berbasis Flutter yang memungkinkan pengguna untuk:
- Mengisi nama sebelum memulai kuis  
- Menjawab pertanyaan pilihan ganda  
- Melihat hasil skor akhir  


Aplikasi ini dirancang dengan tampilan responsif yang bisa menyesuaikan ukuran layar perangkat (desktop, tablet, maupun smartphone).
